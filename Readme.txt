# Однопользовательский морской бой
### Как использовать:
Способы ввода стартовых данных в программу:
1. Через консоль.
Сначала нужно ввести 2 целых числа, разделенных пробелом, отвечающих за размеры поля(количество столбцов и строк)
Далее вводится 5 целых чисел разделенных пробелом - количество кораблей каждого типа, от наименьшего к наибольшему.
После этого вводится 1 целое число - количество торпед, если число равно 0 режим считается выключенным
И в конце вводится 1 символ "y" или "n", который обозначает включен ли режим восстановления кораблей.

Все запросы ввода из консполи сопровождаются информацией о формате и сути ввода.
2. Через аргументы коммандной строки
Сначала вводятся 8 целых чисел через пробел:
- количество строк
- количество столбцов
- количество кораблей длины 1
- количество кораблей длины 2
- количество кораблей длины 3
- количество кораблей длины 4
- количество кораблей длины 5
- количество торпед
И через пробел вводится символ "y" или "n" для включения или выключения режима восстановления кораблей соответственно.
